# Happy Aprilazio Portfolio

Static portfolio website for GitHub Pages.

## How to edit
- Replace `assets/profile.jpg` to change profile photo.
- Replace `assets/CV_Happy_Aprilazio.pdf` to update CV.
- Edit project links in `index.html`.
- Edit visual style in `styles.css`.

## GitHub Pages
Upload all files to your repository, then enable GitHub Pages from Settings > Pages > Deploy from branch > main > /root.
